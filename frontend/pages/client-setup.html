ArubaOS-S Switch - HP/Aruba 2530

        configure terminal
          radius-server host 192.168.4.8 key "Secure Key"
          aaa authentication port-access eap-radius
          aaa port-access authenticator 7
          aaa port-access authenticator 7 client-limit 5
          aaa port-access authenticator active
          aaa port-access mac-based 7
          aaa port-access mac-based 7 addr-limit 5
          aaa port-access mac-based 7 unauth-vid 666
          aaa port-access mac-based addr-format no-delimiter
        write memory

Use no-delimiter MAC format so MAC auth usernames match Radiusaurus. Example: f67c8ebda773

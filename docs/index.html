What it manages

Radiusaurus turns the operational pieces of a FreeRADIUS deployment into a simple management workflow.

Identity and access

Create and manage RADIUS users, MAC-auth devices, groups, VLAN assignments, and client secrets.

Certificates

Generate, export, download, revoke, and inspect client certificates from the web UI.

Generated configs

Generate clients.conf, EAP, SQL, queries.conf, default site, inner-tunnel, and request logging configs from templates.

Health dashboard

Check FreeRADIUS status, config validation, database connectivity, and recent authentication results.

Advanced editor

Use the built-in raw config editor for whitelisted FreeRADIUS files when you need emergency manual edits.

Portable customer setup

Keep customer-specific values in settings.json and sensitive credentials in .env for cleaner deployments.

Quick install

curl -fsSL https://raw.githubusercontent.com/zoutbot-cpu/radiusaurus/main/scripts/quick-install.sh -o quick-install.sh
sudo bash quick-install.sh

The installer prompts for the database user/password, Radiusaurus admin account, public URL, certificate details, and downloads the latest release archive.

Deployment workflow

Install Radiusaurus
Run the quick installer on a clean Ubuntu LTS server.
Review settings
Open Settings and verify public URL, RADIUS IP/DNS, certificate settings, SQL settings, and FreeRADIUS paths.
Generate configs
Use Settings to generate all managed FreeRADIUS configuration files from templates.
Test before reload
Run the built-in FreeRADIUS config test. Radiusaurus can show stdout/stderr directly in the UI.
Reload safely
Reload FreeRADIUS only after validation passes.
